This project performs Exploratory Data Analysis (EDA) on job listings scraped from RemoteOK.com.
The goal is to clean the dataset, remove inconsistencies, fix missing values, visualize insights, and prepare the data for further analysis or ML applications.

Project Overview

This EDA project includes:

Loading raw job data (remoteok_jobs.xlsx)

Cleaning missing values, duplicates, and inconsistent records

Creating a processed dataset (remoteok_jobs_cleaned.xlsx)

Generating meaningful visualizations for insights

Understanding hiring trends across locations, job types, and posting dates

This project is suitable for:
1 Data Science learning
2 Portfolio and resume projects
3 College submissions
4 Dashboard / ML model preparation
